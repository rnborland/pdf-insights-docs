PDF-Insights Documentation

Turn PDFs into searchable, programmable AI assistants.

PDF-Insights helps users query documents naturally and helps developers integrate document intelligence into their own applications through a simple API.

Choose Your Path

Whether you want to use PDF-Insights directly or integrate it into software, start with the path that fits you best.

📘 Usage Instructions

For users who want to upload PDFs, ask questions, and use the platform directly.

🧠 Developer API

For developers who want to create an account, generate an API key, upload PDFs, and query them programmatically.

What PDF-Insights Can Do

  • Upload PDF documents and query them in natural language
  • Turn the same PDF into different assistants using system prompts
  • Support developer workflows through API-based integration
  • Provide structured responses suitable for applications and tools
