# API Examples

This directory contains working examples for using the PDF-Insights.ai API.

## Examples

- streamlit_catalog_assistant.py  
  A generic Streamlit app that turns a PDF catalog into a sales, support, or product recommendation assistant.

## Getting Started

1. Get an API key from:
   https://users.pdf-insights.ai

2. Install dependencies:
   pip install streamlit requests

3. Run the example:
   streamlit run streamlit_catalog_assistant.py

## Notes

- Upload a product catalog, guide, or price list PDF
- Ask product, technical, or quote-style questions
- Customize behavior using the system_prompt
